6 Holes

6Holes is a two player game developed using React,phoenix framework,Elixir. In this project we have used Backup agent, GenServer which allows the user to close the browsers and join the game again from the same point at whic he left. The game starts with 2 players, each having 6 cards, 3 cards in each row respectively. The players are able to see only two cards at the start of the game. The column two cards are opened initially for both the players.The goal of each player is to get minimum possible score. The score is calculated based on each column. If two cards in a column match, the score evaluates to zero, the sum of scores of each column contributes to an overall score. The player can device stratergies to discard a card which is of no use to the opponent. The middle card can be swapped at any moment unless the columns does not count to a score of zero. Thus at each consecutive turn the player tries to get minimum possible card, also the side cards belonging to first and second column can be swapped only if they are not opened, once the cards are unveiled they cannot be swapped.

Memory Game

The Memory game is web application developed using Phonenix framework, Elixir, React.The game comprises of a 4X4 grid of tiles. Here each tile is associated with a value: letter in range A-H.Clicking on tile exposes the associated letter.On clicking on second tile if it matches with the first then both are marked completed. Else both the tiles are hidden again.The number of clicks to match all the pairs (complete all the tiles) should be tracked. Less clicks is a better score at the game.It has a mechanism (e.g. a button) to restart the game at any time.
